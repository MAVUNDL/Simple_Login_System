# CustomTkinter Login Application

This is a simple login application built with `CustomTkinter` in Python.

## Features

- A login form with fields for email and password.
- A 'Login' button and a 'Continue With Google' button.
- The application window is not resizable and has a fixed size of 800x600 pixels.
- The application uses an image as the background.

## How to Run

1. Ensure you have Python installed on your system.
2. Install the `CustomTkinter` and `Pillow` packages using pip:

```bash
pip install customtkinter Pillow
```

3. Replace the image paths in the script with the paths to your actual images.
4. Run the Python script:

```bash
python script_name.py
```

Replace `script_name.py` with the name of the Python file.

## Usage

1. Enter your email and password into the respective fields.
2. Click the 'Login' button to log in.
3. Click the 'Continue With Google' button to log in with Google.
